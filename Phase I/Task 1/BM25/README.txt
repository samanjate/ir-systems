The BM25 is run on inverted_index.txt file found in the Indexing CACM folder.
it takes the query input from the calm.queries.txt file.

Please make sure to add these two file and the bm25.py file in your current directory 
of your Python IDLE and use the command:

  execfile(‘bm25.py’) 

to run the file.

The output of the program will be the cacm_queries_bm25.txt file.
