The CSVSM is run on inverted_index.txt file found in the Indexing CACM folder.
it takes the query input from the calm.queries.txt file.

Please make sure to add these two file and the csvsm.py file in your current directory 
of your Python IDLE and use the command:

  execfile(‘csvsm.py’) 

to run the file.

The output of the program will be the cacm_queries_csvsm.txt file.
